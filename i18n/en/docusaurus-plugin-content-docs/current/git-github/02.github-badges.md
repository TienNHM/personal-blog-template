# GitHub Badges

In this article, I will guide you how to create GitHub Badges. This is a new feature of GitHub, it allows you to create badges to display on your profile.

## Introduction

GitHub Badges is badges displayed on your profile. These badges can display information such as: number of followers, number of repositories, information about the number of visits to your profile, ...

## Guide to create GitHub Badges

Today, you can easily create GitHub Badges through available tools. These tools will help you create badges easily.

Shields.io is a tool that helps you create badges easily. This website provides badges with many different colors and styles. Some popular types of badges are: `social`, `version`, `downloads`, `rating`, `size`, `activity`, `custom`,...

### 3. [github-profile-trophy](https://github.com/ryo-ma/github-profile-trophy)
